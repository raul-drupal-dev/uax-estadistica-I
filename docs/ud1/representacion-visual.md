## Objetivo

:frame_with_picture: Conocer gráficos básicos: barras, pastel, histograma, diagrama de caja (boxplot) y cuándo usar cada uno.

## Resumen rápido

- Variables cualitativas: gráfico de barras, gráfico de sectores.
- Variables cuantitativas: histograma, densidad, boxplot.

## Ejemplo

Si medimos alturas (cm) en una muestra, usaremos histograma para ver la distribución y boxplot para visualizar mediana, cuartiles y posibles outliers.
